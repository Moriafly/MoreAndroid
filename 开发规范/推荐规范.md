# 推荐规范

## 命名规范
### 1. MoreAndroid 设定的控件命名规范

    一个实现打开功能的按钮：btnOpen，采用小驼峰命名，不适用下划线
    
    | 控件       | 对应 |
    | --------- | ---- |
    | Button    | btn  |
    | CardView  | cv   |
    | ImageView | iv   |
    | RecycleView  | rv   |
### 2. 代码命名
    采用小驼峰，如

    ```kotlin
    fun main() {

    }
    ```
